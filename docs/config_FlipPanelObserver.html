Config: FlipPanelObserver

A controller for Flipswitch panels. Individual panels have to be grouped to this object using GroupId. Once all panels have been pressed, this controller will activate a switch, spawn a Power Star or both. A total of 32 objects can be grouped in SMG1 whereas the limit is 128 in SMG2.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Completion Cutscene Time: How long the game will be interrupted after activating all panels. One additional second will be added by the game.

    Obj_arg7IntegerBothFalse

    Power Star ID: The scenario ID of the Power Star to be spawned. 0 will look for the first scenario entry. -1 does not register a Power Star.
