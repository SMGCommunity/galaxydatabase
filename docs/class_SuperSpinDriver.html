Class: SuperSpinDriver

A large star-shaped ring that launches the player from one point to another.

Arguments

NameTypeGamesRequired?DescriptionExclusives?Values

Flight Time: The amount of time the flight takes to complete (in frames). Default 300.

      Obj_arg1FloatBothno

      Shadow Drop Distance: The distance to drop the shadow. Default -1.

          Obj_arg2BooleanBothno

          Don't Grab Mario: If set, the Launch Star won't pull the player towards it when they get near.

              Camera Count: The number of Cameras to be read from the camera file.

                  No Appear Jingle: If enabled, the usual Launch Star appear jingle will not play.

                      Landing Rotation: The rotation that the player lands at. (NOTE: Needs further research as the rotation angle seems inconsistent)

                              SpinDriverPathStorage Index: The index of this launch star's path in the internal SpinDriverPathStorage class. This Arg is read by both games but only functions in SMG1.
