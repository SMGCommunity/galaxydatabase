Class: GlobalWireGravity

A gravity area that attracts objects towards the gravity points laid out on a path. The gravity points will be selected at equal intervals.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Point Count: How many gravity points to calculate, excluding the first and last points on the path. Default is 20.

    Switches

    NameGamesRequired?Description
    SW_ABothFalse

    If used, the gravity will be activated when this switch is activated.

    SW_BBothFalse

    If used, the gravity will be activated when this switch is not activated.
