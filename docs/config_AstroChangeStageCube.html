Change Type: Defines where this area should take you to.

  • 0: Move to Comet Observatory (Dome Exit)
  • 1: Move to Gateway Galaxy

  • 3: Move to Library
  • 4: Move to Comet Observatory (Library Exit)
  • 5: Move to Observatory Dome

Spawn ID: If Obj_arg0 is set to 0, this value goes unused.

    • 2: Fountain
    • 3: Kitchen
    • 4: Bedroom
    • 5: Engine
    • 6: Garden
