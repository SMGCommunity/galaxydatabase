Config: LuigiNPC

A non-playable variant of Luigi that can be talked to. In the case of LuigiEvent, it appears by default unless Obj_arg0 is set. If the object is LuigiNPC, Obj_arg0 will always yield Normal animation behavior, except for when it is set to 0, in which case, Luigi will be scared and can spawn a Power Star.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG1False

Behavior: The animation/behavior to be used. The animation will always be "Normal" for LuigiNPC. Some behaviors have quirks that change based on certain conditions, such as 2 on LuigiEvent having an upward launch behavior when SW_APPEAR / SW_SLEEP is used, turning into Normal behavior upon landing.

  • -1: Normal (Always appeared, cannot spawn Power Star)
  • 0: Normal
  • 1: Stuck to a wall, then Normal
  • 2: Panicked
