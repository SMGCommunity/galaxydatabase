Config: LuigiNPC

A non-playable variant of Luigi that can be talked to. This object will only appear under certain conditions, which is determined by the value of Obj_arg0. This can be bypassed by using SW_APPEAR / SW_SLEEP.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG1False

Behavior: The animation/behavior to be used. If the object is called LuigiEvent, 2 will be added to the value. For any value greater than 0, the object will only appear under certain conditions, unless bypassed with SW_APPEAR / SW_SLEEP.

  • -1: Normal
  • 0: Afraid
  • 1: Normal (Observatory)
  • 2: Normal (Good Egg)
  • 3: Stuck to a wall, then Normal (Honeyhive)
  • 4: Panicked (Battlerock)
