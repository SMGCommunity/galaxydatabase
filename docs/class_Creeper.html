Class: Creeper

A large, usually curved, flower that the player can swing along by shaking the Wiimote. Once the player reaches the end of the flower, it will cause them to jump off. The speed and bending are configurable.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0FloatBothFalse

Normal Bending Factor: Specifies how much the plant bends when grabbed by the player. Default is 100.

    Obj_arg1FloatBothFalse

    Horizontal Speed: Specifies the horizontal flinging speed. Default is 25.

      Obj_arg2FloatBothFalse

      Vertical Speed: Specifies the vertical flinging speed. Default is 40.

        Obj_arg3FloatBothFalse

        Bee Bending Factor: Specifies how much the plant bends when grabbed with the Bee power-up. Default is -1 which causes Obj_arg0 to be used instead.
