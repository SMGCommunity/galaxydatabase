Config: TicoCoin

One of the main collectibles in a Galaxy. There should only be one in every proper Galaxy. TubeSliderTicoCoin is a variant to be used on tube slides. It needs to be linked to a proper TubeSlider instance. TubeSliderTicoCoin has a far clipping distance of 300 meters.
