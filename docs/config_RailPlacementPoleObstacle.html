Config: RailPlacementPoleObstacle

Creates paintbrushes that are arranged on a line. The rotation of all paintbrushes is affected by gravity.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0FloatSMG2False

Interval: How much space there is between each paintbrush. Default is 100. The value should be positive.

    Obj_arg1IntegerSMG2False

    Color Type: What paintbrush model to be used.

    • -1: Green (PoleObstaclePartsA)
    • 1: Red (PoleObstaclePartsB)
    • 2: Blue (PoleObstaclePartsC)
    • 3: Yellow (PoleObstaclePartsD)
