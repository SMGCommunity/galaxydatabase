Config: StaffRollDemoObj

Creates and registers several objects as well as the scrolling text that appear during the ending credits. The objects need to be linked to it using GeneratorId. These are StaffRollSky, StaffRollDecorateFaceShip, StaffRollDecorateComet and PeachCastleNightSky.

Gets activated when the credits start playing. Gets deactivated again after the credits played.
