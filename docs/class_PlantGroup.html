Class: PlantGroup

A group of decorative plants that can appear in many different shapes. In SMG2, some objects of this class use ShapeModelNo to derive the proper model to be loaded. The plants attach to the nearest collision below them. If no collision is available, the individual plant may be placed at the origin of the galaxy.
