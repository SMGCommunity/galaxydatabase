Config: TamakoroJumpGuidanceArea

An area that displays a layout showing how to jump with a Star Ball.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG2False

Power-Up Condition: The required power-up state condition to enable the tutorial.

  • -1: No power-up required
  • 0: Player has Cloud power-up
  • 1: Player has Boo power-up
  • 2: Player rides Star Ball

Switches

NameGamesRequired?Description
SW_APPEARBothFalse

The object will appear when this switch is activated.

SW_ABothFalse

If used, the tutorial will be shown only if this switch is activated. Can be used in combination with SW_B.

SW_BBothFalse

If this and SW_A are used, the tutorial will be shown only if SW_A is activated and SW_B is deactivated. Has no use if SW_A is not used.

SW_AWAKEBothFalse

The object will become visible and resume movement when this switch is activated.
