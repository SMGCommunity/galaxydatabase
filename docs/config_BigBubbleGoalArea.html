Config: BigBubbleGoalArea

An area that pops any Big Bubble inside it, optionally activating its SW_A. This area renders a dark blue billboarded circular meter at the center of this area that turns yellow as it gets filled.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG1False

Meter Capacity: The capacity of the meter, in relation to Big Bubble size. Default is 1, which will activate the switch with the default Big Bubble size.
