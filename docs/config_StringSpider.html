Config: StringSpider

A spider enemy that hangs from a thread of webbing. When the player is nearby, it will drop down and swing back and forth on their threads to damage the player. They can be defeated using most of the player's attacks, explosions and objects like Stretch Plants. They can drop a Coin or three Star Bits.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0FloatBothFalse

Thread Length: The length of its thread. Default is 300.

    Obj_arg1IntegerSMG1False

    Swing Axis: The axis that it swings along. If set to 2, it will swing along its own Z axis.

    • -1: Any axis
    • 2: Z axis
    Obj_arg2IntegerSMG1False

    Shadow Length: Sets the shadow drop length. Default is -1 which causes it to use a spherical shadow instead.
