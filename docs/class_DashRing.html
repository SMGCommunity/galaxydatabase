Class: DashRing

A red ring that causes the player to gain a short speed boost when entered while swimming or flying with the Flying power-up.

Arguments

NameTypeGamesRequired?DescriptionExclusives?Values
Obj_arg0IntegerBothno

Acceleration Time: The time in frames it takes to reach the maximum boost. Default is 300.

          Speed Boost: Speed boost multiplied by 100. Default is 250.
