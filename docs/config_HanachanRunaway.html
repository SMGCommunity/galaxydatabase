Config: HanachanRunaway

A variant of a Wiggler intended to be used on tube slides. It needs to be connected to a proper TubeSlider instance using GeneratorId in order to be created. Its Obj_args specify how it gets positioned on the slide. Unlike its normal variants, it cannot be defeated.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG2False

Forwards Speed: How fast it moves along the TubeSlider's path. The actual speed value needs to be multiplied by 10. Default is 100.

    Obj_arg1IntegerSMG2False

    Sidewards Speed: How fast it revolves around the TubeSlider's path. The actual speed value needs to be multiplied by 10. Default is 20.

      Obj_arg2IntegerSMG2False

      Number of Body Parts: How many body parts it consists of. Minimum is 1, default is 5.

        Obj_arg3BooleanSMG2False

        Is Angry?: If set, it will turn red and move around more aggressively.

          Obj_arg5FloatSMG2False

          Initial Angle: The angle around the tube slide at which the object gets positioned initially.
