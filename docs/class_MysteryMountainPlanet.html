Class: MysteryMountainPlanet

A collision-less cake planet that is wrapped inside a present box. It uses the two animations BoxVanish and CakeVanish, which cause the box and cake to vanish if SW_A is activated. If the player is within 3000 units while the object is being initialized, the cake will automatically vanish.
