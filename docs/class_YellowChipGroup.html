Class: YellowChipGroup

A controller which activates SW_A when all Star Chip (YellowChip) objects linked to it by the obj_arg0 value are collected. It is recommended to have groups of 5 Star Chips. This controller is also responsible for displaying the Star Chip Counter layout at the top of the screen.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothTrue

Group ID: The ID of this group for Star Chips to link to.

    Obj_arg2IntegerBothFalse

    Star Chip Counter Display Range: The distance the player has to be within of this object for the Star Chip Counter to be displayed.

    • -1: = Always display
    Obj_arg3BooleanBothFalse

    Re-activate When Previously Activated: If checked, re-activates SW_A upon a soft reload (ex: after a death) if already activated before.
