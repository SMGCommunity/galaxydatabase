Class: CapsuleCage

A locked cage that can be opened by activating its SW_B switch.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg4IntegerSMG2False

Riddle SFX Type: Specifies what sound effect to play after the cage gets opened.

  • -1: No special sound effect
  • 1: Play RiddleSE
Obj_arg6BooleanSMG2False

Play Star Chance Music?: If set, it will play the STM_STAR_CHANCE_3 music once it gets opened.

    Obj_arg7BooleanSMG2False

    Use Alternate Model?: Tries to load the model CapsuleCageNoKeyHole if enabled. This model does not exist anymore, though.
