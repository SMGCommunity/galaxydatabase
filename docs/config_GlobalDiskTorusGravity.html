Config: GlobalDiskTorusGravity

A simple gravity area that pulls objects in its range towards a torus/ring-shaped volume.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0BooleanBothFalse

Disable Bottom?: If enabled, only the upper portion of the disk will have gravity.

    Obj_arg1IntegerBothFalse

    Valid Surfaces: The valid surfaces to attract objects to. Both are enabled by default.

    • -1: Both sides
    • 0: No sides
    • 1: Inside only
    • 2: Outside only
    Obj_arg2FloatBothFalse

    Tube Radius: The tube's radius, not to be confused with the major radius of the entire torus. Default 0.

      Switches

      NameGamesRequired?Description
      SW_ABothFalse

      If used, the gravity will be activated when this switch is activated.

      SW_BBothFalse

      If used, the gravity will be activated when this switch is not activated.

      SW_AWAKEBothFalse

      Enables rendering and resumes updates when this switch is active.
