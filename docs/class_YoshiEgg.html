Class: YoshiEgg

An egg that can be cracked open to reveal Yoshi, the player's rideable dinosaur pal.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0BooleanSMG2False

Nest Type: If set, only the nest will be shown at the start of the level. If another Yoshi returns to its egg, this nest will become availible to use as a respawn point for Yoshi. This setting will be ignored if SW_APPEAR is used.

  • -1: Egg always enabled
  • 1: Egg initially disabled

Switches

NameGamesRequired?Description
SW_APPEARSMG2False

The object will appear when this switch is activated.

SW_ASMG2False

If set, the player will be unable to break open the egg until the switch is activated.

SW_AWAKESMG2False

The object will become visible and resume movement when this switch is activated.
