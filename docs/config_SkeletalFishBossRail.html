Config: SkeletalFishBossRail

Specifies an individual subpath that Kingfin can move on. These also determind when and where the Guard Fishes spawn.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG1True

Rail ID: The ID of this rail. Used by SkeletalFishBoss to decide what path to use for each phase.

    Point_arg0IntegerSMG1True

    Is Guard Spawn Node?: If set, Kingfin will summon his Guards when crossing this point.

    • -1: Do not Spawn Guards
    • 0: Spawn Guards
