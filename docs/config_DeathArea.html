Config: DeathArea

An area that can kill the player and compatible objects when entered.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Death Type: Specifies what objects this area affects.

  • -1: Kill player and compatible objects
  • 1: Kill compatible objects except player
