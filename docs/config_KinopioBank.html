Config: KinopioBank

A Toad that allows the player to store Star Bits that can be shared across save files. He may give players free Star Bits depending on the number of stored Star Bits. An additional Toad (Kinopio) can be linked to it using GeneratorId which will be hidden once this object is set to appear.
