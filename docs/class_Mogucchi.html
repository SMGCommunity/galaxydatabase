Class: Mogucchi

A mole enemy that moves on a set path. The player takes damage by touching it. It can be stunned by Ground Pounding nearby which makes it vulnerable to Spin Attacks. However, it can be defeated with any player attacks, such as fireballs and Koopa Shells.
