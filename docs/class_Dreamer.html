Class: Dreamer

A green TV that displays the player's screen. It can be used to execute some pre-recorded developer gameplay after talking to it which is usually used to teach the player some gameplay elements. The PAD data for it can be found in the pad folder of the galaxy's Ghost archive. It can use optional text when talked to, however, it will default to the text labeled as DreamerCommon000.
