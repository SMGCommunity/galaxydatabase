Config: Meister

The large purple Luma NPC that can be talked to. It registers to various Starship Mario cutscenes and events.

Arguments

Registers Starship Events?: If set, it will register and register to various Starship Mario events. This is only read if the current stage is MarioFaceShipGalaxy.
