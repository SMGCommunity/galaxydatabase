Config: MameMuimuiScorer

A controller for The Chimp's Skating Challenge which sets up the minigame and Gummits. The offsets and sequences of the Gummits are specified in the ActorInfo files PositionList.bcsv and Sequence.bcsv, respectively. The player has 40 seconds to score points. The default high scores are 500 for MameMuimuiScorer and 600 for MameMuimuiScorerLv2. The object needs to be linked to a MameMuimuiAttackMan instance.
