Config: MiniatureGalaxy

A selectable galaxy model used inside Domes. The galaxy to warp to is specified in the name of the object. The conditions for each galaxy to unlock are specified in the GameEventFlagTable.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG1False

Galaxy Type: What type of galaxy this is. Also specifies the model for the object if the galaxy hasn't been unlocked yet. Boss galaxies use model MiniKoopaGalaxy until cleared at least once.

  • 0: Regular galaxy, uses MiniatureGalaxyUnknown
  • 1: Secret galaxy, uses MiniHatenaGalaxy
  • 2: Boss galaxy, uses MiniatureGalaxyUnknownKoopa
