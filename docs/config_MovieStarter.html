Config: MovieStarter

Starts and plays a THP cutscene video which interrupts the current game scene.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerSMG1True

Movie ID: The target movie to play. Using any value not listed will crash the game.

  • 0: PrologueA (As Luigi, this movie gets skipped)
  • 1: PrologueB (Sends you to Grand Star Rescue)
  • 2: FinalBattle, FinalBattleLuigi as Luigi (Sends you to Scenario Select for Bowser's Galaxy Reactor. Can be skipped after watching once.)
  • 3: EpilogueA, EpilogueALuigi as Luigi (Screen stays White afterwards, but gameplay is resumed)
  • 5: EndingA, EndingALuigi as Luigi (Screen stays Black afterwards, but gameplay is resumed)
  • 6: EndingB

Switches

NameGamesRequired?Description

Appears when this switch is activated, making the movie begin.
