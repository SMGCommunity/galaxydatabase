Config: SeaGullGroup

A group of several decorative seagulls. All seagulls will be placed along the specified path, however, the seagulls will not fly along the path. The number of seagulls to be spawned can be specified as well. If the object is SeaGullGroupMarioFace, the seagulls will appear only if the current world that the player resides in is World 1, World 2 or World S.
