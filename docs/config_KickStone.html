Config: KickStone

A small kickable stone. It drops a single Star Bit or a Coin when hit by the player or explosions. The object automatically tries to connect to collision within 300 units.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0BooleanSMG2False

Drops Coin?: If set, it will drop a single Coin when kicked. Otherwise, it will drop a Star Bit.

    Obj_arg1BooleanSMG2False

    Connector Ignores Gravity?: If set, the object's down vector will be used to determine the collision to connect the object to. Otherwise, the object's gravity will be used.

      Obj_arg2BooleanSMG2False

      Don't Connect to Collision?: If set, it won't connect to nearby collision.
