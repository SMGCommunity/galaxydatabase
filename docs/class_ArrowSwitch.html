Class: ArrowSwitch

A rotating switch object that can be activated using Spin Attacks.

Arguments

NameTypeGamesRequired?DescriptionExclusives?Values
Obj_arg0IntegerBothno

Rotation Direction: Specifies the rotation when hit.

    • -1: 180° to right or left
    • 0: 90° to left
    • 1: 90° to right
    • 2: 180° to left
    • 3: 180° to right
    Obj_arg1BooleanBothno

    Activation Mode: If enabled, it cannot be hit again util the SW_A event deactivates again.

        Obj_arg2IntegerBothno

        Shadow Length: Sets the shadow drop length.
