Config: ShootingStar

A shooting star that releases Star Bits when it hits the ground. It will periodically reappear until all of its Star Bits have been collected.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Star Bits per Iteration: How many Star Bits should be released at max per iteration. Default is 5.

    Obj_arg1IntegerBothFalse

    Spawn Delay: The spawn delay of the individual shooting stars in seconds. Default is 240.

      Obj_arg2FloatBothFalse

      Falling Distance: From how far up it will fall down. Default is 2000.

        Obj_arg3IntegerBothFalse

        Number of Total Star Bits: How many Star Bits can appear in total. Default is 5.

          Switches

          NameGamesRequired?Description
