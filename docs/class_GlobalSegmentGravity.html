Class: GlobalSegmentGravity

A gravity area that pulls objects towards the lateral surface of the cylinder.

Arguments

NameTypeGamesRequired?DescriptionExclusives?Values

Edge Type: The type of edges that will be used. Default "Both".

    • -1: = Both
    • 0: = Neither
    • 1: = Bottom Only
    • 2: = Top Only
    Obj_arg1FloatBothno

    Valid degrees: The radial degrees that will have gravity. Starts pointing towards X+, and then rotates counter clockwise. Default 360.0.
