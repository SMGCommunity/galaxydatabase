Config: WaterPlant

A group of decorative seaweed that can appear in many different shapes. The seaweed attach to the nearest collision below them. If no collision is available, the individual plant may be placed at the origin of the galaxy.

Arguments

Number of Seaweed: The number of seaweed to be placed. Default is 16.

    Placement Range: The range in which the seaweed will be placed. Default is 500.
