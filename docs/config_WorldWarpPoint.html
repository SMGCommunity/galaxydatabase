Config: WorldWarpPoint

A decorative portal from the World Map that appears depending on the currently selected point on the World Map. If the object is called WorldWarpPointEntrance, it will appear only if the player is near an entrance portal on the World Map; if the object is called WorldWarpPointExit, it will appear only if the player is near an exit portal on the World Map.
