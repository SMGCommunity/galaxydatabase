Config: ExterminationSwitch

A controller that activates a switch once all linked and compatible objects have been killed. Individual objects have be linked to this object using GeneratorID. ExterminationPowerStar spawns a Power Star after killing all objects. It also uses a counter layout to keep track of killed objects.
