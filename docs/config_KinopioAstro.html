Config: KinopioAstro

A variant of a Toad that is used in the Comet Observatory. It features a wide range of different behaviors and equipment. It can attach to Sling Pods.

Color Frame: The color frame to be used. If set to be Purple, it will carry the Postman Bag.

  • -1: Blue
  • 1: Green
  • 2: Purple
  • 3: Red
  • 4: Yellow
Obj_arg2IntegerSMG1False

Behavior: The animation/behavior to be used.

  • -1: Default
  • 0: Spinning
  • 1: Breakdancing
  • 2: Faster spinning
  • 3: Far talking range
  • 4: Connects to collision below (Broken in SMG2)
  • 5: Swimming
  • 6: Digging
  • 7: Sleeping
  • 8: Spawns Power Star
  • 9: Guarding
  • 10: Excited/jumping
  • 11: Scared
  • 12: Holds Star Bit
  • 13: Panicking while running
Obj_arg4IntegerSMG1False

SW_APPEAR Behavior: How it functions when its SW_APPEAR switch gets triggered. This setting is ignored if Obj_arg2 is set to 4.

  • -1: Appears immediately
  • 0: Gets launched from box
