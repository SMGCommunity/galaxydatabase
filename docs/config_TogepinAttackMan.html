Config: TogepinAttackMan

The monkey NPC who will prompt the player to play a scoring minigame where one has to defeat Pinheads with the Rock power-up. The minigame controller (TogepinScorer) needs to be linked to it using GeneratorId. It will register a Power Star, however, this requires a properly configured Scenario entry. A complex dialogue with flow nodes needs to be set up as well.
