Config: PhantomTorch

A flame that burns the player and other compatible objects upon contact. It can be extinguished using a Spin Attack.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Item Type: Specifies the item to be spawned after putting out the fire.

  • -1: Coin
  • 0: Star Bit
  • 1: Nothing
Obj_arg1BooleanBothFalse

Decorative?: If set, the flame won't burn anything and it can't be extinguished.

    Switches

    NameGamesRequired?Description
    SW_APPEARBothFalse

    The object will appear when this switch is activated.

    SW_DEADBothFalse

    Gets activated when it dies.

    SW_AWAKEBothFalse

    The object will become visible and resume movement when this switch is activated.

    Setup

    NameGamesRequired?Description
    GroupBothFalse

    Groups other objects of the same kind. Special sound effects will be played when extinguishing the flames in this group. The limit of objects in a group is 32.
