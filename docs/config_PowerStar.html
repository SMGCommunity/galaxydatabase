Config: PowerStar

The main collectible in most levels. A Power Star usually represents one specific scenario. When collecting it, a short cutscene will be played. If this is a Grand Star and the current galaxy is VsKoopaLv3Galaxy, Peach will appear during the Star Get cutscene.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Power Star ID: The scenario to which this Power Star belongs. Should be a valid ID.

    Obj_arg1BooleanBothFalse

    Appearance Gravity?: If set, gravity will affect how the Power Star moves during its appearance cutscene.

      Obj_arg2BooleanBothFalse

      Ignores Gravity?: If set, its pose won't be affected by gravity and is instead specified by the initial rotation.

        Obj_arg3BooleanBothFalse

        Elliptic Cylinder Shadow?: If set, an elliptic cylinder will be used as its shadow model instead of an ellipsoid.

          Obj_arg4BooleanBothFalse

          Appear Nearest Point?: If set, the Power Star will appear at the PowerStarAppearPoint closest to the player. Otherwise, the Power Star's original position will be used if the player is closer to it than to the other points.

            Obj_arg5FloatSMG2False

            Shadow Length: Sets the shadow drop length. Default is 1000.
