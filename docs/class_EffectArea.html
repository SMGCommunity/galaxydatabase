Class: EffectArea

An area in which most of the player's dust cloud particles can be replaced with yellow flower petals.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Change Type: How the particles are adjusted

  • -1: Default
  • 1: Flower petals (Obj_arg0 has to be set to 0)
Obj_arg1IntegerBothFalse

Particles Type: The type of particles to use.

  • -1: Dust clouds
  • 0: Flower petals (requires Obj_arg0 = 1)
