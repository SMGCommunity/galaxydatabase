Config: FlexibleSphere

An object capable of shrinking and growing between two sizes.

Arguments

Maximum Scale: Maximum scale of the object, multiplied by 10000. Default is 10000.

    Minimum Scale: Minimum scale of the object, multiplied by 10000. Default is 1000.

      Obj_arg2IntegerSMG1False

      Scale Rate: Scaling rate of the object, multiplied by 10000. Default is 20.

        Obj_arg3BooleanSMG1False

        Vanish After Shrinking?: If set to 1, the object disappears after reaching the minimum scale.

          Obj_arg4IntegerSMG1False

          Useless: Read but never used for anything.
