Config: SurprisedGalaxy

A portal that warps the player to a specific galaxy. It has a far clipping radius of 200 meters. The galaxy to warp to is specified in the name of the object, unless it is "SurprisedGalaxy", in which case it warps to SurfingLv1Galaxy. If the object is SurpTamakoroExLv2Galaxy, SurpCubeBubbleExLv2Galaxy or SurpSurfingLv2Galaxy, it will only appear once all three Green Stars have been collected, and if the object is SurpPeachCastleFinalGalaxy, it will only appear once the player has collected 120 Stars with both Mario and Luigi. Tthese checks can be bypassed with SW_APPEAR / SW_SLEEP.
