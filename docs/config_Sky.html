Config: Sky

A decorative background model that is meant to be placed around entire galaxies. It follows the camera's position, so it's not possible for the player to reach the end of the model, unlike atmosphere objects.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Mirror Reflection Type: Specifies whether to use a reflection model or not.

  • -1: Don't use reflection model
  • 0: Use reflection model
