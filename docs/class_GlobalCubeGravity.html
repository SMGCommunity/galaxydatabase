Class: GlobalCubeGravity

A gravity area for a cube-shaped planet that allows movement on all six faces.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

X Axis Faces: Defines which sides on the X Axis of the cube are active. Default is "Both Sides"

  • -1: = Both Sides
  • 0: = No Sides
  • 1: = X+ Side only
  • 2: = X- Side Only
Obj_arg1IntegerBothFalse

Y Axis Faces: Defines which sides on the Y Axis of the cube are active. Default is "Both Sides"

  • -1: = Both Sides
  • 0: = No Sides
  • 1: = Y+ Side only
  • 2: = Y- Side Only
Obj_arg2IntegerBothFalse

Z Axis Faces: Defines which sides on the Z Axis of the cube are active. Default is "Both Sides"

  • -1: = Both Sides
  • 0: = No Sides
  • 1: = Z+ Side only
  • 2: = Z- Side Only

Switches

NameGamesRequired?Description
