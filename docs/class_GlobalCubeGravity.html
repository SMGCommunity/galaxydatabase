Class: GlobalCubeGravity

A cube-shaped gravity area that pulls objects towards the cube's surface. Each of the cube's faces can be manually toggled for further customizability. The transition around edges and corners is smooth. The object's scale defines the actual size of the cube.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Valid X Axis Faces: Defines which sides on the X axis of the cube are valid. Both are enabled by default.

  • -1: Both sides
  • 0: No sides
  • 1: X+ side only
  • 2: X- side Only
Obj_arg1IntegerBothFalse

Valid Y Axis Faces: Defines which sides on the Y axis of the cube are valid. Both are enabled by default.

  • -1: Both sides
  • 0: No sides
  • 1: Y+ side only
  • 2: Y- side Only
Obj_arg2IntegerBothFalse

Valid Z Axis Faces: Defines which sides on the Z axis of the cube are valid. Both are enabled by default.

  • -1: Both sides
  • 0: No sides
  • 1: Z+ side only
  • 2: Z- side Only
