Config: AssemblyBlock

Remains at a resting position above the object's actual position. Once the player gets in its range, it will move to its actual position. This class supports ShapeModelNo models. The timed activation requires additional effect data under the UniqueName "Blur".

NameTypeGamesRequired?DescriptionValues
Obj_arg0FloatBothFalse

Activation Range: Specifies the activation range to the player's position. If set to -1, the model's diameter is used instead.

    Obj_arg1IntegerBothFalse

    Activation Time: If Obj_arg7 is set, this specifies the timer until the block disappears again.

      Obj_arg7BooleanBothFalse

      Is Timed?: If enabled, the block will be activated for a set amount of time only.
