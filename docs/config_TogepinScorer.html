Config: TogepinScorer

A controller for The Chimp's Bowling Challenge which sets up the minigame whereas the individual Pinheads need to be linked to it using GeneratorId. The controller itself needs to be linked to a TogepinAttackMan instance. The player needs to score 5000 points in order to win.
