Class: DarknessRoomDoor

A pair of closed doors that can be opened.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0BooleanBothFalse

Collision Mode?: If set and SW_A is active, it will synchronize the collision's validity with the switch's state.

    Switches

    NameGamesRequired?Description
    SW_APPEARBothFalse

    The object will appear when this switch is activated.

    SW_ABothFalse

    If Obj_arg0 is set and this switch is used, it will keep the collision validated as long as this switch is active.

    SW_BBothFalse

    Makes the doors open once activated.

    SW_AWAKEBothFalse

    The object will become visible and resume movement when this switch is activated.
