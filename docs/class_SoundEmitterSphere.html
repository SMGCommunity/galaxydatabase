Class: SoundEmitterSphere

An area that starts playing a sound effect when entered. This always has a spherical range.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothTrue

Sound ID: The Sound ID to use. In SMG1, these are hardcoded. In SMG2, more can be added by adding entries to the ActionSound bcsv

  • 0: SE_AT_LV_MAGMA_AMBIENT
  • 1: SE_OJ_LV_FLOATING_SAND
  • 2: SE_AT_LV_WIND_DESERT
  • 3: SE_AT_LV_MAGMA_WIND_VOLCANO
  • 4: SE_AT_LV_KOOPA_FLOAT_SAND
  • 5: SE_AT_LV_WIND_RABBIT_MAZE
  • 6: SE_AT_LV_MAGMA_WIND_MID_HIGH (SMG2 Only)
  • 7: SE_AT_LV_MAGMA_WIND_STRONG (SMG2 Only)
Obj_arg1IntegerBothFalse

Sound Check Point: The point to use for calculating stereo sounds.

  • 0: Use the position the camera is looking at
  • 1: Use the position of the game's Microphone (presumably this is at the position of the camera)
