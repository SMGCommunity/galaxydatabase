Class: BreakableCage

A destroyable obstacle that can be destroyed using Koopa Shells or explosions like those caused by Bob-ombs. Most of its variants can contain an item. Except for Star Bits and Power Stars, the items are only cosmetic. If the item is set to Power Star, the object also requires a proper setup in the galaxy's Scenario file.
