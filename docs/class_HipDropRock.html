Class: HipDropRock

A rock that can be destroyed with a Ground Pound attack. It can spawn some items after destroying it.

Arguments

NameTypeGamesRequired?DescriptionValues
Obj_arg0IntegerBothFalse

Item Type: The item to be spawned after destroying it.

  • -1: No item
  • 0: 1 Coin
  • 1: Star Bits
Obj_arg1IntegerBothFalse

Number of Star Bits: The amount of Star Bits to be dropped. Default is 6. Requires Obj_arg0 set to 1.

    Obj_arg7BooleanBothFalse

    Hide Remains?: If set, the smaller rocks that remain after destroying it will be hidden.
